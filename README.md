# Interactive Stock Market Analysis Dashboard

## Overview
This project presents an interactive dashboard for analyzing historical stock price data of major technology companies. It combines exploratory data analysis with an interactive Streamlit-based interface to visualize trends, returns, and correlations in stock market data.

## Objective
To analyze and visualize time-series stock price data and provide interactive tools for exploring trends, returns, and relationships between major tech stocks.

## Stocks Analyzed
- Apple (AAPL)
- Amazon (AMZN)
- Google (GOOGL)
- Microsoft (MSFT)

## Features
- Closing price trend visualization
- Moving averages (10, 20, 50 days)
- Daily return analysis
- Monthly, quarterly, and yearly resampling
- Correlation heatmap between stocks
- Interactive dashboard using Streamlit

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly
- Streamlit

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/interactive-stock-market-dashboard.git
2. Navigate into the project directory:
```bash
   cd interactive-stock-market-dashboard
```
3. Install dependencies:
```bash
  pip install -r requirements.txt
```

4. Run the Streamlit dashboard:
```bash
streamlit run stock_dashboard.py
